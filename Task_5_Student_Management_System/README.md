# Task 5 - Student Management System

This project is developed as part of the CodSoft Java Internship.

## Description
A console-based Java Student Management System that allows users to manage student records efficiently. The system supports adding, editing, searching, deleting, and displaying student information using Object-Oriented Programming concepts and file handling for data persistence.

## Features
- Add new student records
- Edit existing student information
- Search student by roll number
- Display all students
- Remove student records
- File handling for permanent storage
- Menu-driven user interface
- Input validation for correct data entry

## How to Run
javac StudentManagementApp.java  
java StudentManagementApp
