# Task 2 - Student Grade Calculator

This project is developed as part of the CodSoft Java Internship.

## Description
A console-based Java application that calculates the total marks, average percentage, and grade of a student based on marks obtained in multiple subjects.

## Features
- Accepts marks (out of 100) for multiple subjects
- Calculates total marks
- Computes average percentage
- Assigns grade based on performance
- Simple and user-friendly console interface

## Grade Criteria
- A : 90% and above  
- B : 75% – 89%  
- C : 60% – 74%  
- D : 40% – 59%  
- F : Below 40%

## How to Run
javac GradeCalculator.java  
java GradeCalculator
