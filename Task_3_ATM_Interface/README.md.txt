# Task 3 - ATM Machine

This project is developed as part of the CodSoft Java Internship.

## Description
A console-based Java ATM Machine simulation that allows users to perform basic banking operations such as withdrawing money, depositing money, and checking account balance using Object-Oriented Programming concepts.

## Features
- Withdraw money with balance validation
- Deposit money
- Check current account balance
- Menu-driven user interface
- Input validation and user-friendly messages

## How to Run
javac ATMMachine.java  
java ATMMachine
